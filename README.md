# vue中央氣象局月平均地面觀測站資料

### 串接中央氣象局Api將每月平均地面觀測站的數據資料使用vue-chart呈現出來!

### demo https://jacky6450.github.io/Meteorology-Monthly-Average-Observation/

### 資料項目
* 氣壓
* 氣溫
* 相對溼度
* 雲量
* 日照時數
* 降水量
* 風速
