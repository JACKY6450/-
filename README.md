# vue中央氣象局月平均地面觀測站資料

##串接中央氣象局Api將每月平均地面觀測站的數據資料使用vue-chart呈現出來!

###資料項目
* 氣壓
* 氣溫
* 相對溼度
* 雲量覆蓋
* 日照時數
* 累積降雨量
* 風速
